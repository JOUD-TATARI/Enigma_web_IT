القيمة المتوقعة:

هي متوسط القيمة التي يفترضها المتغير العشوائي على المدى الطويل أي متوسط مجتمع القياسات اي يصف اين يتركز التوزع الاحتمالي, تعطى رياضيا بالعلاقة:

E ( x ) = ∑ X P ( X = x )

كما أن المعادلة التالية:

E ( x 2 ) = ∑ X 2 P ( X = x )

تلزمنا في حساب التباين

التباين:

هو مقياس مدى انحراف قيم البيانات عن المتوسط وبالتالي يعطي توصيف كافي عن شكل التوزيع، ويعطى رياضياً بالعلاقة:

V ( x ) = E ( X 2 ) − [ E ( x ) ] 2

خواص القيمة المتوقعة:

1]-

E(X1+X2)=E(X1)+E(X2)

2]-

∀a∈R ;a=const⟹E(a)=a

3]-

E(a.X)=a.E(x)

4]-

E(ax∓b)=aE(X)∓b

خواص التباين:

1]-

V(X1+X2)=V(X1)+V(X2)

2]-

∀a∈R ;a=const⟹V(a)=0

3]-
V ( a . X ) = a 2 . V ( x )
4]-
V ( a x ∓ b ) = a 2 V ( X ) ∓ V ( b ) ; V ( b ) = 0

المتغيرين المستقلين:

نقول عن متغيرين عشوائيين انهما مستقلين إذا كانت القيمة المتوقعة لجدائهما تساوي جداء القيمة المتوقعة لكل منهما أي يساوي: E(X.Y)=E(X).E(Y) وبشكل آخر:

E [ x . y ] = ∑ x i y i P ( x = x i , y = y j )
