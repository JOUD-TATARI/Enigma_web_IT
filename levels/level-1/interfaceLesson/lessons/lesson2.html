العمليات على المجموعات

الاجتماع والتقاطع والفرق والمتمم:

الاجتماع:

هو مجموعة العناصر التي تنتمي الى A أو B (المشتركة وغير المشتركة)

A ∪ B = { x ∈ Ω | x ∈ A ∨ x ∈ B}

التقاطع

هو مجموعة العناصر التي تنتمي الى A وB معاً (العناصر المشتركة)

A ∩ B = { x ∈ Ω | x ∈ A ∧ x ∈ B}

الفرق :

مجموعة العناصر التي تنتمي الى A ولا تنتمي الى B (ويسمى المكمل النسبي).

A-B={ x ∈ Ω | x ∈ A ∧ x ∉ B}

المتمم (المكمل المطلق):

يرمز له ب A متمم المجموعة A وهو مجموعة العناصر التي لا تنتمي الى A ولكنها تنتمي الى Ω.

A={x |x ∈ Ω ∧ x ∉ A}
