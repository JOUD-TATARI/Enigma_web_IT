توزيع بواسون وتقريب ثنائي حد الى بواسون

تجربة بواسون:

هي تجربة عشوائية تهتم بمعدل النجاحات خلال فترة زمنية معينة او منطقة مكانية قد تتراوح الفترة الزمنية بين ثانية الى سنوات والمنطقة المكانية قد تكون حيز المكاني (كتاب، واحدة مساحة، حجم) ويعتبر من التوزيعات الهامة التي تعبر عن عدد الحوادث النادرة الظهور لفترة زمنية معينة او منطقة محدودة.
أمثلة:
1]- عدد الزبائن الواردين إلى مكتب بريد كل 3 دقائق.
2]- عدد حوادث السير على طريق زراعي في أسبوع.
3]- عدد الأخطاء الطباعية في كتاب
4]- عدد المكالمات الهاتفية الواردة الى مقسم الجامعة كل ع10 دقائق.
5]- عدد الكريات الحمراء في عينة الدم
6]- عدد القطع التالفة في الإنتاج الكلي لظاهرة ما

بفرض x متغير عشوائي يعبر عن عدد النجاحات عند اجراء تجربة بواسون ويحقق الشروط.
1]- احتمال النجاح في هذه الفترة الزمنية ثابت وكذلك الفشل.
2]- معدل النجاحات خلال الفترة الزمنية أو المنطقة المكانية. محدود وقدره λ

λ = n × p

3]- مع تضاؤل الفترة الزمنية أو المكانية يصبح احتمال النجاح ضعيف جدا ويقترب من الصفر مقابل زيادة احتمال الفشل واقترابه من الواحد.
4]- عدد مرات تكرار التجربة مستقلة
عندها نقول ان x يخضع لتوزيع بواسون:

x ~ pois ( λ )

توزيع برنولي:

P ( X = x ) = λ x x ! × e − λ ; x = 0 , 1 , 2 , 3 ⋯

E ( x ) = λ , V ( x ) = λ

تقريب التوزيع ثنائي الحد إلى توزيع بواسون:

x متغير عشوائي يخضع الى توزيع ثنائي الحد بالمعلمتين (n,p) اذا كانت n كبيرة بقدر كافٍ واحتمال النجاح p صغير بقدر كافٍ يتم رد التوزيع الى بواسون بالمعلمة λ=n*p.

اختبار مفاجئ

معدل عدد حوادث السيارات عند إحدى التقاطعات هو 3 حوادث أسبوعياً والمطلوب:

1]-

أوجد احتمال عدم وقوع أي حادث عند تلك الإشارة في أسبوع ما

p ( x = 0 ) = e − 3
p ( x = 0 ) = e 3
p ( x = 3 ) = e 3
p ( x = 3 ) = e − 3
2]-

أوجد احتمال وقوع حادثتين أو أقل في الأسبوع

p ( x ≤ 2 ) = 8.5 × e − 3
p ( x ≤ 2 ) = − 8.5 × e − 3
p ( x ≤ 2 ) = 5.3 × e − 3
p ( x ≤ 2 ) = − 5.3 × e − 3
3]-

ما هي خواص هذا التوزيع

E(x) = V(x) = 3

E(x) = 3 ,V(x) = 2

E(x) = 2 ,V(x) = 3

E(x) ≠ V(x)
